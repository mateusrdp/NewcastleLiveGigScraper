# Formats

Summaries are of the form "Main band @ Venue".
Relevant information for the description are:
- actual descriptions
- price (add a Price: $<price> in the description. Eg: "Price: $15.20")
- links to more info (add ad More info: <link> in the description. Eg: "More Info: https://link.to.my/event")
- links to tickets (add a Tickets: <link> in the description. Eg: "Tickets: https://link.to/ticket/shop")
- if there are both a long and a short link, the short one is preferred
- only a single "@ Venue" should be kept per event. Anything from the 2nd '@' (inclusive) should be discarded to keep the summaries short (Ie: "Mat @ Some Pub @ Some Pub" -> "Mat @ Some Pub")
- The first letter of each word in the summary should be capitalised. Stop words and every other character should be lowercase (eg: Temple of the Stag @ Stag and Hunter)
- Venue names should omit the words "the" and "hotel" (eg: The Stag and Hunter Hotel -> Stag and Hunter) to keep summaries short
